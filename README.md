# Personal Portfolio

A modern personal portfolio built with React, Vite
and Tailwind CSS to showcase my projects, skills 
and contact information. 

##  Features

- Responsive Design
- Clean UI
- React Router Navigation
- Skills Section
- Projects Section
- Contact Section
- Resume Download

##  Tech Stack 

- React
- Vite 
- Tailwind CSS
- React Router
